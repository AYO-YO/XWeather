# XWeather

基于聚合数据API完成的一款无广告天气软件

本分支基于`Python Flask`框架开发API完成，不再使用`chaquopy`作为内置`Python`完成数据解析。

API开源地址：<https://gitee.com/ayo_yo/XWeatherAPI>
